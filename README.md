+ 源码解读
+ 相关资料/blog的汇总

### **:tada:** 最有效的资料

学习一个开源系统，什么资料最有效？**官网文档！官方文档！官网文档！**

因为官方文档是社区团队共同编辑和检视的，它所表达的信息将是最准确的、最权威的，其他网上的文档只能作为补充而不能替代官方文档的作用。

所以学习Apache DolphinScheduler之前，请先确保自己已经看完了官网文档：
